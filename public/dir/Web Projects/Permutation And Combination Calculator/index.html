About


Factorial

A function that multiplies a number by every number below it.


Circular Permutation

The total number of ways in which n distinct objects can be arranged around a fix circle.


Distinguishable Permutation

The number of ways of arranging ‘n’ items, out of which ‘p’ are alike, ‘q’ are alike and ‘r’ are alike.


Permutation

The action of changing the arrangement, especially the linear order, of a set of items.


Combination

A selection of a given number of elements from a larger number without regard to their arrangement.
